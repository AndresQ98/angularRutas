{{ titulo }}

Básicamente es la forma que tiene angular para permitir mostrar un contenido de forma dinámica y no algo estático. Es la forma de comunicación entre la parte visual (HTML) y la parte lógica(typescript).

Un ejemplo claro es lo que se está usando en esta aplicación, para cada sección eh declarado en el .ts un título único y en el html llamo a esa variable titulo entre doble llaves.
